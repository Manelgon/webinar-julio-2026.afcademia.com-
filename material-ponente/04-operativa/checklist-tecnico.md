## CONFIGURACIÓN DE ZOOM (detalle)

| Opción | Cómo activarla |
|--------|---------------|
| Sala de espera | Zoom → Mis reuniones → Editar → Seguridad → Sala de espera ✓ |
| Grabación en nube | Zoom → Mis reuniones → Editar → Grabación → Grabar en la nube ✓ |
| Chat de participantes | Durante la reunión → Chat → … → Permitir a los participantes chatear con: Todos |
| Compartir pantalla | Durante la reunión → Seguridad → Compartir pantalla: anfitrión y co-anfitriones |
| Co-anfitrión | Hacer co-anfitrión al asistente nada más entre: clic en su nombre → Hacer co-anfitrión |

## PLAN DE CONTINGENCIA TÉCNICA

| Problema | Acción inmediata |
|----------|-----------------|
| Se cae internet | Activar hotspot del móvil. Reconectar Zoom. Decir: "Volvemos en 30 segundos." |
| Se cae Zoom | Reconnectar. Asistente dice en chat: "Reconectando, en 1 minuto." |
| Falla TextExpander demo | Transición verbal: "Os muestro el resultado directamente" → slide con texto del snippet |
| Falla Make demo | "Os muestro la hoja pre-rellenada que preparé ayer" → Google Sheets |
| Claude va lento | "Os muestro los resultados que obtuve ayer con el mismo prompt" → documento guardado |
| Fallo de audio (los participantes no oyen) | El asistente escribe en el chat "¿Nos oyes?" → ponente comprueba micro → si no se resuelve, reconectar Zoom |
| Ruido de fondo inesperado | Silenciar micro, esperar a que pase, retomar |
