Cómo usar el resultado en el módulo Gmail · Send an Email

Alternativa más sencilla: En lugar de parseJson(), añade un módulo Tools → Parse JSON entre el OpenAI y el Gmail. Ponle como Input {{B.choices[].message.content}} y Make extrae subject y body como variables independientes sin necesidad de funciones.
